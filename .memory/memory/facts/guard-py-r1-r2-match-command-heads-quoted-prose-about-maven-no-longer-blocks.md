Since PR #3424, .claude/hooks/guard.py strips multi-line quoted strings, bash heredocs, and PowerShell here-strings as data, then requires mvn/allure to be the executable at the head of a command segment (env-var prefixes and timeout/npx wrappers are skipped). A gh pr create --body describing a Maven test command no longer trips R1; real invocations after &&, ;, |, newlines still block. Self-test: python3 .claude/hooks/guard.py --self-test (54 cases).
