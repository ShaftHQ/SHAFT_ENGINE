When the Codacy Static Code Analysis check fails on a ShaftHQ/SHAFT_ENGINE PR, the exact findings (pattern id, severity, file:line, message) are available without auth: curl https://app.codacy.com/api/v3/analysis/organizations/gh/ShaftHQ/repositories/SHAFT_ENGINE/pull-requests/<PR>/issues -- parse .data[].commitIssue. Much faster than guessing from the dashboard link. Typical gate: fails on >=1 medium (Warning) severity issue; Info-level CodeStyle alone also flips it when alerts threshold is exceeded.
