To screenshot HTMLHelper/ReportHtmlTheme report templates without running a full suite: extract the Java text blocks with a python script, substitute placeholders mirroring the Java .replace chain (verify no ${...} leftovers), write the HTML to a temp dir, then shaft-mcp driver_initialize CHROME -> browser_set_window_size -> browser_navigate file:///... -> browser_take_screenshot -> driver_quit. browser_take_screenshot requires a workspace-relative outputPath (e.g. target/evidence/x.png); absolute paths outside the MCP workspace are rejected. Useful when the Claude in-app browser pane times out. Also: git worktree add into a deep temp path fails with 'Could not reset index file' (MAX_PATH) - use a short path like C:/Users/Mohab/IdeaProjects/docs-wt-<n>.
