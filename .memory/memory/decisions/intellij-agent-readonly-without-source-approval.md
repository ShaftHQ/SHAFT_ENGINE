SHAFT IntelliJ Assistant no-source Agent mode is allowed for default local CLI runtimes only when SHAFT can enforce a read-only/suggestion-only path: Codex uses `--sandbox read-only` with SHAFT MCP tool approval config, Claude uses plan permission mode, and Copilot uses ask mode. Custom Agent commands without source-edit approval remain rejected because SHAFT cannot enforce read-only execution for arbitrary commands. Browser-only Agent prompts should not be nudged to enable source edits unless the prompt also names source/code artifacts.
