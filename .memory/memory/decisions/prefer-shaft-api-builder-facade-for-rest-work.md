Future REST work should prefer SHAFT.API and RequestBuilder.perform(). Legacy RestActions helpers, DriverFactory.getAPIDriver, and RequestBuilder.performRequest remain deprecated bridges until a deliberate major cleanup. GraphQL requests are built through SHAFT.API.sendGraphQlRequest(...), extended with normal RequestBuilder methods such as addHeader(...), and executed with perform(). Query parameters should rely on RestAssured and session URL preparation instead of manual query string mutation.
