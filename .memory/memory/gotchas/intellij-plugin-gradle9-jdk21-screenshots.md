`shaft-intellij` uses the IntelliJ Platform Gradle plugin, which requires Gradle 9+ and works reliably with JDK 21 in Codex. If the default Gradle/JDK pair fails before tests, install/use Gradle 9 and set `JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64`. For UI PR screenshots, run `ShaftPluginScreenshotRendererTest` with `-Dshaft.intellij.screenshotDir=../shaft-engine/src/main/resources/modular-era-feature-catalog`; it refreshes the full IntelliJ plugin screenshot gallery, not only one PNG.
