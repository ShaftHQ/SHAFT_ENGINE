ShaftProjectServiceTest (shaft-mcp) asserts exact step/locator strings from the bundled example templates in shaft-engine/src/main/resources/examples (e.g. SampleFeatureFile.feature). A template-only commit does not trigger the shaft-mcp CI job, so the broken assertion only surfaces when a later commit touches shaft-mcp. When changing example templates, grep shaft-mcp/shaft-intellij tests for asserted template strings in the same commit (bitten in PR #3718: DuckDuckGo->Wikipedia swap).
