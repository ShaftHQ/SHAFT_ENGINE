When recording the SHAFT IntelliJ IDEA plugin on Windows, both Gradle `runIde` with the cached JetBrains JBR and an installed IntelliJ disposable profile can trigger a Windows Firewall prompt for `OpenJDK Platform binary` before the project opens. Computer-use automation must not choose Windows Security permissions or change firewall rules. Preflight the selected IntelliJ runtime once before starting ffmpeg and have the user manually dismiss/approve the prompt; only then start the real MP4 capture.
